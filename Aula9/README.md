Nesta aula falaremos sobre tomadas de decisoes ou melhor, Estruturas Condicionais.

A linguagem C, possui tres estruturas condicionais:

# 1
if (condicao)
{
    bloco de codigo;
}

# 2
if (condicao)
{
    bloco de codigo a ser executado;
}
else
{
    bloco de codigo a ser executado;
}

# 3
switch(expressao)
{
    case constante1:
        //codigo a ser executado
        break;
    default:
}
